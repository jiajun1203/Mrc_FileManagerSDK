# FileManagerSDK 文件管理SDK
IOS 文件管理，支持：
### 1. 文件拷贝，
### 2. 文件剪贴，
### 3. 文件重命名
### 4. 删除文件
### 5. 拷贝，剪贴 完成/失败 及总数量进度反馈
